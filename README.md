WPF-приложение диспетчера процессов Windows

ProcessManager — настольное WPF-приложение для мониторинга и управления процессами операционной системы Windows в реальном времени.
Программа получает сведения о запущенных процессах через системный API .NET, отображает их основные характеристики и предоставляет пользователю инструменты управления: изменение приоритета, завершение процесса и настройку привязки к ядрам процессора (CPU affinity).
Приложение реализовано с использованием архитектурного паттерна MVVM, асинхронного обновления данных и модульного тестирования, что обеспечивает высокую производительность, расширяемость и стабильность работы.

Цель лабораторной работы
Разработать WPF-приложение для мониторинга и управления процессами операционной системы Windows с использованием технологий платформы .NET, архитектурного паттерна MVVM и асинхронного программирования.

Как использовать:
  Запустите приложение
  Список процессов загрузится автоматически
  Выберите процесс в таблице
  Используйте кнопки управления:
    Kill — завершить процесс
    Change Priority — изменить приоритет
    CPU Affinity — настроить ядра
Список процессов обновляется автоматически в реальном времени.
